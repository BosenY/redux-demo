# redux demo 

> A React.js demo   

> Use react-redux write a Increase Component
## Build Setup

``` bash
# install dependencies
npm install
# or 
yarn install

# 客户端 localhost:8080
npm start

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run test
```
